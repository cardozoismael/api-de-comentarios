# api-de-comentarios

es un proyecto donde en cada endpoint tenes que tener la authorizacion (token) para ver,crear,modificar o eliminar la informacion que esta en la carpeta "comentarios".
todos los endpoints requiereren token para acceder,asegurate de incluirlo en el header :
"Authorization: Bearer <tu_token>"

# instalacion
para usar necesitarias intalar: express y jsonwebtoken

puedes usar mi proyecto atravez de postman ,usando el archivo "MyCollection.json"
